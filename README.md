# OIMG Vite Plugin

Optimize SVG images

## Install

Install

```bash
npm i vite-plugin-psvg -D # pnpm add vite-plugin-psvg -D
```

Add it to `vite.config.js`

```ts
// vite.config.js
import OIMG from 'vite-plugin-psvg'

export default {
  plugins: [
    OIMG()
  ],
}
```
